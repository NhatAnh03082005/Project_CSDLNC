# Pets Module

Module quản lý thú cưng của khách hàng.

## Chức năng

- Thêm/sửa/xóa thú cưng
- Xem danh sách thú cưng
- Xem lịch sử khám bệnh
- Xem lịch sử tiêm phòng

## Files

- `pets.routes.js` - API endpoints
- `pets.controller.js` - Request handlers
- `pets.service.js` - Business logic
- `pets.repository.js` - Database queries
- `pets.validator.js` - Validation rules

## Thuộc tính Thú cưng

- Tên, loại, giống
- Ngày sinh, giới tính
- Tình trạng sức khỏe
- Liên kết với khách hàng
