# Customers Module

Module quản lý thông tin khách hàng.

## Chức năng

- Xem & cập nhật thông tin cá nhân
- Xem thông tin hạng thành viên
- Xem điểm loyalty
- Xem lịch sử dịch vụ & hóa đơn

## Files

- `customers.routes.js` - API endpoints
- `customers.controller.js` - Request handlers
- `customers.service.js` - Business logic
- `customers.repository.js` - Database queries

## Liên kết với modules khác

- `membership/` - Quản lý hạng thành viên & điểm
- `invoices/` - Lịch sử hóa đơn
- `appointments/` - Lịch hẹn
